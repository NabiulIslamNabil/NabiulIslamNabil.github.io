# Nabiul Islam Nabil Portfolio

Live deployment: [Nabiul's Portfolio](https://nabiulislamnabil.github.io/) on github

Live deployment: [Nabiul's Portfolio](https://nabiulislamnabil.netlify.app/) on netlify

A personal portfolio website built to showcase projects, experience, skills, and contact information.

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Radix UI
- GSAP
- Lenis
- Three.js
- React Router
- Lucide React
- Sonner
- Zod

## Tools Used

- ESLint
- PostCSS
- Autoprefixer
- TypeScript
- Vite plugin for React
- Vite plugin for GLSL

## Project Structure

- `app/src/pages` - page-level routes
- `app/src/sections` - main portfolio sections
- `app/src/components` - reusable components and UI primitives
- `app/src/hooks` - custom hooks
- `app/src/lib` - utility helpers

## Development

From the `app` directory:

```bash
npm install
npm run dev
```

## Build

From the `app` directory:

```bash
npm run build
```
